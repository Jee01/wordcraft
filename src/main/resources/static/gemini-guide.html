⚡ WordCraft AI ← 설정으로 돌아가기
Gemini API Key 발급 방법
Google AI Studio에서 무료로 Gemini API Key를 발급받을 수 있습니다.
WordCraft AI는 현재 Gemini 2.5 Flash 무료 티어만 지원합니다.
💡 무료 티어 제한 안내
Gemini 2.5 Flash 무료 티어는 분당 요청 수(RPM) 및 일일 요청 수(RPD)에 제한이 있습니다. 개인 학습 용도로는 충분하지만, 단시간에 많은 단어를 생성할 경우 일시적으로 제한될 수 있습니다.
1
Google AI Studio 접속
아래 링크에서 Google AI Studio에 접속합니다. Google 계정으로 로그인이 필요합니다.

https://aistudio.google.com
2
API Key 메뉴 이동
로그인 후 왼쪽 사이드바에서 「Get API key」 버튼을 클릭합니다.
3
API Key 생성
「Create API key」 버튼을 클릭합니다.
프로젝트 선택 팝업이 나타나면 기존 프로젝트를 선택하거나 새 프로젝트를 생성합니다.
4
API Key 복사
생성된 API Key가 표시됩니다. 키는 AIza로 시작하는 형식입니다.
복사 버튼을 눌러 클립보드에 복사합니다.
AIzaSy••••••••••••••••••••••••••••••••••
5
WordCraft AI 설정에 입력
설정 → AI API Key 관리의 Google (Gemini) 항목에 붙여넣고 「API Key 저장」을 클릭합니다.

이제 단어장 만들기에서 AI 자동 생성 기능을 사용할 수 있습니다.
⚠️ 보안 주의사항
API Key는 브라우저 localStorage에만 저장되며 서버에는 전달되지 않습니다. 단, API Key가 외부에 노출되면 타인이 사용할 수 있으므로 공용 PC에서는 사용 후 반드시 삭제해 주세요. 키가 유출된 경우 Google AI Studio에서 즉시 삭제하고 새로 발급받으세요.
