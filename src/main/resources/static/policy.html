서비스 소개

WordCraft AI는 비영리·학습 목적으로 운영되는 AI 기반 영단어 학습 플랫폼입니다. 이 서비스는 수익을 목적으로 하지 않으며, 개인 학습 및 포트폴리오 용도로 제공됩니다.

수집하는 정보

  • 이메일 주소 — 회원 식별 및 비밀번호 찾기 목적
  • 닉네임 — 서비스 내 표시 이름
  • 비밀번호 — 암호화(BCrypt)하여 저장, 원문은 보관하지 않음
  • 학습 데이터 — 단어장, 테스트 결과 등 사용자가 직접 입력한 데이터
  • Google 계정 정보 — Google 소셜 로그인 이용 시 이메일·닉네임 수신

수집 목적

  • 회원 가입·로그인 및 본인 확인
  • 단어장·테스트 등 학습 서비스 제공
  • 비밀번호 찾기 이메일 발송

보관 및 삭제

  • 회원 탈퇴 시 수집된 모든 개인정보 및 학습 데이터를 즉시 삭제합니다.
  • 관계 법령에 따라 보존 의무가 있는 경우 해당 기간 동안만 보관합니다.

제3자 제공 및 외부 서비스

  • Google OAuth — 소셜 로그인 시 Google 인증 서버를 경유합니다.
  • Gemini AI API — 단어 자동 생성 기능 이용 시 사용자가 직접 입력한 API Key로 Google Gemini API를 호출합니다. API Key 및 입력 단어는 서버에 저장되지 않습니다.
  • AWS — 서버 및 파일 저장소로 AWS 인프라를 사용합니다.
  • 수집된 개인정보는 위 서비스 운영 목적 외 제3자에게 제공하지 않습니다.

쿠키 및 보안

  • 로그인 상태 유지를 위해 HttpOnly 쿠키를 사용합니다. JavaScript에서 접근할 수 없어 XSS 공격으로부터 보호됩니다.
  • 테마 설정 등 기능 편의를 위해 localStorage를 일부 사용합니다.
  • 광고·추적 목적의 쿠키는 사용하지 않습니다.

이용약관

  • 서비스는 학습 목적으로만 이용해야 하며, 불법적인 용도로 사용할 수 없습니다.
  • 커뮤니티에 공유하는 단어장의 저작권은 작성자에게 있습니다.
  • 서비스는 사전 공지 없이 변경·중단될 수 있습니다.
  • 비영리 서비스 특성상 데이터 손실에 대한 법적 책임을 지지 않습니다.
