핵심 기능

WordCraft AI가 특별한 이유

🏷️

태그 기반 AI 생성

수능·토익·토플 등 태그를 선택하면 해당 맥락에 최적화된 뜻과 예문을 AI가 자동 생성합니다.

📂

멀티 입력 자동 변환

텍스트 붙여넣기, 이미지 사진, PDF·DOCX 파일 — 기존 단어장을 어떤 형태로든 바로 변환합니다.

🔑

내 AI Key 직접 사용

OpenAI, Claude, Gemini 등 원하는 AI API Key를 브라우저에만 저장. 서버는 중계만 수행합니다.

🌐

커뮤니티 공유

내가 만든 단어장을 공개하거나 다른 사람의 단어장을 복사해 바로 학습할 수 있습니다.

📝

다양한 테스트

뜻 맞추기, 빈칸 채우기, 플래시카드, 오답 노트까지. 결과 통계로 취약 단어를 파악합니다.

🔤

품사 자동 분류

명사·동사·형용사·숙어·복합명사 등 AI가 자동으로 품사를 분석하고 태깅합니다.
